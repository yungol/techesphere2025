La ilusión de la conversación

La IA no tiene memoria

send POST /api/chat
Usuario: "Hola, ¿cómo estás?"
IA: "¡Hola! Estoy bien, ¿en qué puedo ayudarte?"
refresh
send POST /api/chat
Usuario: "¿De qué hablamos antes?"
IA: "No tengo información sobre conversaciones anteriores"
refresh
code Responsabilidad del Desarrollador
Cada POST debe incluir:
• Todo el historial de la conversación
• Contexto relevante
• Instrucciones del sistema

Cada llamada a la API es un nuevo comienzo.
Es el desarrollador quien debe construir la ilusión de una conversación continua.
El contexto es la clave para una interacción coherente y relevante.
